Detailed Description

Symulator Turnieju Piłkarskiego - moduł główny.

Moduł odpowiedzialny za zarządzanie całym turniejem piłkarskim. Zawiera logikę przepływu turnieju od fazy grupowej do finału.

Author
Sebastian Orman, Michał Nowakowski
